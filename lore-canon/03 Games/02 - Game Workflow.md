# Game Workflow

## Use Games Folder For
- game-specific rules
- dev logs
- prompts
- backlog
- systems
- production notes

## Best Flow
1. If a rule only affects one game, keep it in that game's folder
2. If it changes world truth, update Master Canon too
3. Keep each game's identity protected
4. Do not let one game's mechanics overwrite another

## Important Rule
Shared lore can exist across games.
Shared mechanics should not be assumed.
