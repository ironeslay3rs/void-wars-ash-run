# Void Wars: Oblivion — Core Rules

## Identity
Main game of the IP.
Canon-first.
Lore core always wins.

## Design Direction
Use outside references only as structural inspiration.
Do not let borrowed systems erase Void Wars identity.

## Influence Spine
- Overmortal for progression structure
- AFK Journey for presentation clarity and readable field feel
- Vampire Wars / Darkest Dungeon influence where useful
- Void Wars canon always overrides all influence

## Build Rules
- clean modular layered architecture
- thin route/page files
- logic in proper feature layers
- reusable components
- professional team-safe structure
- M1-safe scope control

## Important Rule
This is not the card game.
This is not the old browser game.
This is not the PvP fighter.
This is not Ash Run.
