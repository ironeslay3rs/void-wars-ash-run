# How To Use This Vault

## Put notes here
- random/raw ideas -> 00 Inbox
- universe truths -> 01 Master Canon
- book work -> 02 Books
- game work -> 03 Games
- manga work -> 04 Manga
- anime planning -> 05 Anime
- live action planning -> 06 Live Action
- visual identity -> 07 Art Bible
- workflow/templates/process -> 08 Production
- brand/public strategy -> 09 Marketing Brand

## Important rule
Logs are not canon.
Only locked canon notes count as canon.

## Best daily workflow
1. Capture idea in Inbox
2. Move it to the right folder
3. Turn stable truths into canon notes or locked rules
4. Keep media-specific work separated
