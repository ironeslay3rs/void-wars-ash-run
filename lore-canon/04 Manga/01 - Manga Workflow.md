# Manga Workflow

## Use Manga Folder For
- chapter plans
- page scripts
- panel layouts
- prompt packs
- final page tracking

## Best Flow
1. Master Canon defines truth
2. Manga decides visual pacing and scene structure
3. Keep page logic in Manga, not in Book folders
4. Keep prompt packs tied to actual page needs

## Important Rule
The manga adapts the IP visually.
It should not casually overwrite canon truth.
