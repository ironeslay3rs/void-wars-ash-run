# Book 04 — Cross Media Value

## Game Value
- economy systems
- currency systems
- Black Market structure
- Chronicle / game-within-game logic
- factional survival mechanics

## Manga Value
- strong Blackcity / Ember Vault / market visuals
- card, currency, and invitation symbolism
- layered social danger

## Anime Value
- high intrigue
- economy and power explained through tension rather than exposition
- dramatic Chronicle entry hook

## Live Action Value
- very adaptable as a dark systems-thriller
- strong world-expansion season material

## Key Strength
The world becomes organized around power, trade, survival, and hidden rules.
