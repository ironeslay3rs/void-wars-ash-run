# Book 01 — Cross Media Value

## Game Value
- early survival loop
- condition system (fatigue, body pain)
- grounding player experience

## Manga Value
- strong visual realism
- lighting contrast
- slow reveal pacing

## Anime Value
- cinematic grounded opening
- strong emotional hook

## Live Action Value
- highly adaptable
- realistic setting
- low budget entry possible

## Key Strength
Human connection before power.
