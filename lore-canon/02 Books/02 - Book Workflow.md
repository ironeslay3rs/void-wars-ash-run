# Book Workflow

## Use Books Folder For
- saga planning
- book outlines
- chapter planning
- scene planning
- quotes
- canon locks
- revisions

## Best Flow
1. Put big truth in Master Canon if it affects the whole IP
2. Put book-specific execution in the correct Book folder
3. Put final structural decisions in Canon Locks
4. Keep rough ideas in Scenes until promoted

## Important Rule
Book folders are for execution of the saga.
Master Canon is for universal truth.
