# Currency System

Currencies in the IP reflect philosophy, scarcity, and power.

## Core Currencies
- Ichor
- Soul Crystals
- Sinful Coin
- Mecha material value systems

## Ichor
Primary Bio-linked value system.
Represents blood, essence, life-force extraction, and adapted biological worth.

## Soul Crystals
Primary Pure-linked value system.
Used for learning, memory, spiritual refinement, and inherited experience.

## Sinful Coin
Primary Black Market currency.
Represents mixed value, survivor exchange, and neutral/inter-sin commerce.

## Mecha Value Systems
Often tied to rare materials, engineered parts, ancient remains, and compatibility resources such as Ironheart.
