# Creator Workflow

## Daily Workflow
1. Capture raw ideas in `00 Inbox`
2. Move real ideas to the correct folder
3. Turn stable truths into canon or locked rules
4. Keep medium-specific execution in that medium

## Weekly Workflow
1. Update `Current Focus`
2. Review Inbox
3. Lock canon changes
4. Update active project notes
5. Review Books / Games / Manga priorities

## Important Rule
Do not leave valuable ideas trapped in Inbox forever.
Process them into the system.
